# Enterprise AI Knowledge System

An AI-powered document management and knowledge retrieval system built using FastAPI, React, ChromaDB and Groq LLM.

## Features

- User Authentication (JWT)
- Upload PDF & TXT Documents
- AI Semantic Search
- ChromaDB Vector Database
- Search History
- Dashboard Analytics
- File Preview
- User Settings
- Dark / Light Theme

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- JWT Authentication
- ChromaDB
- Sentence Transformers
- Groq Llama

### Frontend
- React
- Vite
- Axios
